# Ma0 Y Leidy

Una página con React Router para nuestra boda


## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server with HMR:

```bash
npm run dev
```

Your application will be available at `http://localhost:5173`.

## Building for Production

Create a production build:

```bash
npm run build
```

## Deployment

```
cd build/client
aws s3 sync . s3://ma0yleidy.com
```

Check the s3 site at http://ma0yleidy.com.s3-website-us-east-1.amazonaws.com Real changes may take a few minutes to reflect.
